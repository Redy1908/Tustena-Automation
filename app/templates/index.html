Come funziona

Impostazioni

Configura la API key Tustena e l'URL del feed iCal (Float) prima di utilizzare l'app. I mapping nomi permettono di correggere automaticamente le discrepanze tra i nomi presenti nel calendario e quelli in Tustena. La configurazione viene salvata nel browser e non è necessario reinserirla ad ogni accesso.

Dashboard settimanale

Le allocazioni del feed iCal vengono mostrate settimana per settimana, raggruppate per giorno. Usa le frecce per navigare tra le settimane.

Creazione voucher

Clicca Crea sul singolo voucher per inviarlo a Tustena. L'operazione non può essere annullata: eventuali correzioni vanno effettuate direttamente su Tustena. Dopo la creazione ricordati di inviare il rapportino via mail da Tustena.
